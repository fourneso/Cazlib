ICazExtenderCallbacks Interface
This interface is used by Caz Suite to pass to extensions a set of callback methods that can be used by extensions to perform various actions within Caz. When an extension is loaded, Caz invokes its registerExtenderCallbacks() method and passes an instance of the ICazExtenderCallbacks interface. The extension may then invoke the methods of this interface as required in order to extend Caz's functionality.
